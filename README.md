# gameboy-pico
Software written in MicroPython for the "GameBoy Pico" a Raspberry Pi Pico Project using a 8x8 LED Matrix.
Including the "device drivers" and the games.

## hardware components

- Raspberry Pi Pico
- 8x8 LED Matrix using a max7219 controller
- 4 Buttons
